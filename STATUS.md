# STATUS — lobby_analysis

Last updated: 2026-04-21

## Current Focus

The `scoring` branch has executed the Phase 3 three-state pilot (CA / CO / WY × 3 temp-0 runs × 3 rubrics) and flagged PRI disclosure-law as insufficiently consistent across all three pilot states (11.5%–37.7% disagreement). Root cause: the snapshot corpus is captured with a same-host constraint (state SOS/ethics portal only), which excludes state legislative sites where the actual statute text lives — so disclosure-law items are being scored against the portal's *summary* of the law rather than the law itself. **Phase 4 (50-state scale-up) is paused** pending a methodology calibration: use PRI 2010 human-rater scores + 2010-era statutes as ground truth, iterate the scorer prompt, then apply the calibrated prompt against 2026 statutes. Plan doc at `docs/active/scoring/plans/20260417_pri_ground_truth_calibration.md`; intended to be picked up by a fresh agent on a new branch.
